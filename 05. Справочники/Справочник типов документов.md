Единый источник правды для всех документов, которые могут быть приложены к заявке. Используется:
- в [[00. Исполнение заявки — обзор|исполнении заявки]] ([[01. Исполнение — Платёж через банк|Исполнение — Платёж через банк]], [[02. Исполнение — Платёж через агента|Исполнение — Платёж через агента]], [[03. Исполнение — Закупка|Исполнение — Закупка]]) — для загрузки документов по мере их получения на каждом шаге;
- в [[09. Проверка комплектности и закрытие заявки|Проверка комплектности и закрытие заявки]] — как чек-лист комплектности перед закрытием заявки (без повторной загрузки — только проверка, что все обязательные типы уже загружены).

## Примечание
Список кодов и обязательности — предварительный, основан на текущих процессах. При появлении новых типов документов — добавлять сюда, а не в отдельные заметки по процессам.

См. также [[Бизнес-правила|BR-050…BR-053]] по правилам обязательности и загрузки документов.
