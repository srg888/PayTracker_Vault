# Делегирование полномочий Руководителя

Сквозной процесс, действующий параллельно основному [[01. Жизненный цикл заявки|жизненному циклу заявки]]: позволяет временно передать полномочия Руководителя другому пользователю — обычно Исполнителю.

## Участники
- **Делегирующий** — пользователь с ролью Руководителя.
- **Делегат** — пользователь, получающий временные полномочия (в терминах [[Роли и права]] — "Делегированный руководитель").

## Создание делегирования
Руководитель указывает:
- пользователя-делегата;
- дату и время начала;
- дату и время окончания;
- причину делегирования.

При создании проверяется:
- делегирующий имеет право создавать делегирование;
- делегат активен;
- период корректен;
- отсутствует другое пересекающееся активное делегирование.

**Правило (BR-030):** одновременно может существовать не более одного активного делегирования полномочий Руководителя.

## Действие делегирования
Делегирование активно, если:

```text
start_at <= текущее время <= end_at
```

(**BR-031**). После окончания периода делегирование автоматически перестаёт предоставлять полномочия — никакого отдельного действия не требуется.

Пока делегирование активно, назначенный Исполнитель получает права Руководителя: принимать новые заявки ([[03. Назначение исполнителя|Назначение исполнителя]]), назначать/переназначать исполнителя, отклонять заявки — полный перечень см. [[Матрица прав]] (колонка "Делегированный руководитель", обозначение **D**).

## Досрочный отзыв
Руководитель может досрочно отозвать делегирование (**BR-032**). При отзыве сохраняются:
- дата отзыва;
- пользователь, выполнивший отзыв;
- причина отзыва.

## Ограничение
Делегированный пользователь получает только те полномочия, которые прямо предусмотрены [[Матрица прав|Матрицей прав]] (**BR-033**). Делегирование не изменяет значение роли пользователя в базе данных — `role` пользователя остаётся неизменным, право на действие проверяется дополнительной проверкой "есть ли активное делегирование?" (см. [[Матрица прав]], раздел "Делегирование не меняет роль пользователя").

## Аудит
В [[Аудит|журнал аудита]] попадают: создание делегирования, изменение периода, начало действия, досрочный отзыв, окончание действия. Все действия, совершённые в рамках делегирования, помечаются "от имени Руководителя (делегировано)".

## Пример
```text
Руководитель
    ↓
назначает
    ↓
Исполнитель Иванов
    ↓
полномочия Руководителя
    ↓
01.08.2026 — 15.08.2026
```

В период действия Иванов может выполнять действия, разрешённые [[Матрица прав|Матрицей прав]] для делегированного Руководителя. После 15.08.2026 эти права автоматически прекращаются.
