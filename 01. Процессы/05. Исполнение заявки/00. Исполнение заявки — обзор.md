# Исполнение заявки — обзор

Третий этап [[01. Жизненный цикл заявки|жизненного цикла заявки]]. Заявка в статусе `В работе`. Конкретный процесс исполнения зависит от типа заявки:

- **Платёж** — сначала [[00a. Согласование условий исполнения (Платёж)|Согласование условий исполнения]] (способ, комиссия, курс — принятие/отказ Заказчика), затем исполнение выбранным способом:
  - [[01. Исполнение — Платёж через банк|Исполнение — Платёж через банк]]
  - [[02. Исполнение — Платёж через агента|Исполнение — Платёж через агента]]
- [[03. Исполнение — Закупка|Исполнение — Закупка]]
- [[04. Исполнение — Консультация|Исполнение — Консультация]]

Во время исполнения Исполнитель может запросить у Заказчика уточнение — см. [[06. Уточнение|Уточнение]] (цикл `В работе ⇄ Уточнение`; уточнение может быть запрошено и раньше, ещё до назначения Исполнителя — см. BR-042).

Документы по заявке загружаются Исполнителем по мере получения на каждом шаге — какие документы существуют и какие обязательны, см. [[Справочник типов документов]]. Повторной загрузки на следующем этапе ([[09. Проверка комплектности и закрытие заявки|Проверка комплектности и закрытие заявки]]) не требуется — там только сверка комплектности уже загруженного.

По завершении исполнения — [[07. Подтверждение исполнения Заказчиком|Подтверждение исполнения Заказчиком]].
