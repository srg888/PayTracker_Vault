# Исполнение — Платёж через банк

Один из способов исполнения [[01. Поля заявки — Платёж|платёжной заявки]], часть [[00. Исполнение заявки — обзор|исполнения заявки]]. Наступает после [[00a. Согласование условий исполнения (Платёж)|согласования условий]] (способ, комиссия, курс приняты Заказчиком).

## Шаги
- Выбор банка-плательщика.
- Прямой платёж, либо платёж через мастер-агента.

### Прямой платёж (без мастер-агента)
1. Платёж выполняется банком напрямую по реквизитам получателя, указанным в заявке.
2. Исполнитель загружает платёжное поручение или СВИФТ (см. [[Справочник типов документов|ПБ-01]]) → это и есть [[07. Подтверждение исполнения Заказчиком|подтверждение исполнения]], после которого заявка уходит на подтверждение Заказчику.
3. После того как Заказчик подтвердил исполнение, Исполнитель предоставляет закрывающий документ — сведения о валютной операции (СВО, см. [[Справочник типов документов|ПБ-02]]).

### Платёж через мастер-агента
Если платёж через мастер-агента — проверить, предусматривает ли договор оплату от третьих лиц:
  - если нет — подготовить дополнительное соглашение (ДС) к договору (см. [[Справочник типов документов|ПБ-00]]);
  - подготовка агентского договора (чекбокс, загрузка документа);
  - поручения (чекбокс, загрузка документа);
  - акта-отчёта (чекбокс, загрузка документа);
  - СВИФТ (чекбокс, загрузка документа);
  - СВО (чекбокс, загрузка документа).

Если платёж идёт через мастер-агента, комплект документов далее совпадает с комплектом [[02. Исполнение — Платёж через агента|Исполнение — Платёж через агента]] (коды ПА-01…ПА-10 в [[Справочник типов документов]]).
