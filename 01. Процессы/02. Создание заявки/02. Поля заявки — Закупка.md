# Поля заявки — Закупка

Часть процесса [[00. Создание заявки|Создание заявки]]. Общие правила подачи и неизменяемости полей — там же.

> Отдельная спецификация полей формы для заявки на закупку в исходных материалах не зафиксирована — этот раздел собран по [[ER модель|ER-модели]] (`PURCHASE_REQUESTS`) и по шагам процесса [[03. Исполнение — Закупка|Исполнение — Закупка]]. Требует уточнения и дополнения полным списком полей формы, аналогично [[01. Поля заявки — Платёж|Поля заявки — Платёж]].

## Известные поля (по ER-модели и процессу исполнения)
- Название заявки, подразделение заказчика — общие поля `REQUESTS`.
- Компания-покупатель — выбирается из справочника [[Компании-покупатели]].
- Способ оплаты:
  - через платёжного агента (если договор предусматривает оплату от третьего лица — чекбокс), см. [[02. Исполнение — Платёж через агента|Исполнение — Платёж через агента]];
  - прямой платёж от компании-покупателя.
- Ожидаемая дата поставки.
- Форма для загрузки файлов — те же лимиты, что и в [[01. Поля заявки — Платёж|Поля заявки — Платёж]] (до 50 файлов, до 50 Мб на файл, до 500 Мб суммарно; pdf, word, excel, графика, архивы).
