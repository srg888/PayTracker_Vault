# Поля заявки — Консультация

Часть процесса [[00. Создание заявки|Создание заявки]]. Общие правила подачи и неизменяемости полей — там же.

## Поля, заполняемые Заказчиком
- ФИО заказчика
- Подразделение заказчика (Юрлицо)
- Название заявки
- Описание задачи / вопрос
- Ожидаемый срок ответа
- Форма для загрузки файлов — можно загрузить несколько файлов, лимит на 1 файл — 50 Мб, лимит на все файлы — 500 Мб. Максимальное количество файлов — 50. Форматы: pdf, word, excel, распространённые графические форматы, файлы архивов.
