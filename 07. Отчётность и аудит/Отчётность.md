## Базовые отчёты
- Количество заявок за период, в разрезе:
  - каждого Исполнителя ([[03. Назначение исполнителя|Назначение исполнителя]])
  - каждого Заказчика
  - каждого агента ([[Агенты]])
  - подразделения / юрлица Заказчика
  - типа заявки (платёж / закупка / консультация)

## Дополнительные отчёты
- Суммы платежей за период в разрезе валют, юрлиц заказчика, агентов (на основе фактической суммы в рублях на дату исполнения, см. [[08. Фактическое исполнение платежа|Фактическое исполнение платежа]] и [[01. Поля заявки — Платёж|Поля заявки — Платёж]]).
- Просроченные заявки — факт исполнения vs "Ожидаемый срок исполнения".
- Среднее время обработки заявки по типу заявки (от "Новая заявка" до "Закрыта", см. [[Статусы заявки]]) — для контроля SLA.
- Количество заявок, прошедших через статус "Уточнение" (см. [[06. Уточнение|Уточнение]]) — индикатор качества исходных данных от Заказчиков.
- Полная история одной заявки (выгрузка из [[Аудит]]) — для разбора спорных ситуаций.

## Экспорт
- Экспорт в xlsx (Excel) — реестр заявок за период с возможностью фильтрации по всем разрезам выше.
- Импорт справочных данных (агенты, компании-покупатели, курсы валют) из xml/csv.

## Правила (см. [[Бизнес-правила]])
- **BR-090.** Отчёты формируются на основании сохранённых данных операций и истории системы.
- **BR-091.** Для финансовых отчётов используются фактически исполненные суммы и сохранённые значения курсов. Текущий курс валюты не должен изменять историческую отчётность.
